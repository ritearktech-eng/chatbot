This is a demo page showing how to embed the Prime Chatbot widget into any website.

Paste the following code before the closing </body> tag of your website:

<script src="http://localhost:3000/widget.js" data-company-id="951438e3-69ca-49c4-832f-aa3fa13e3eae"> </script>

Look at the bottom right corner to see the widget in action!
